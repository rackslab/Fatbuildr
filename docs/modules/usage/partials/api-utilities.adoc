=== Utilities

==== Software Version

`GET /version`

*No permission action required.*

Returns the version of Fatbuildr server.

===== Responses

[cols="{tbl-responses-cols-specs}"]
|===
|Code|Description|Content Type|Data Type

|200
|Operation successful
|`text/plain`
|_string_
|===

.Examples
[%collapsible]
====
Request:

[source,shell]
----
$ curl -X GET http://localhost:5000/version
----

Response:

[source]
----
Fatbuildr v2.0.0
----
====
